# WorkGuard — Workforce Management System (Client Application)

## Overview
WorkGuard is a production-inspired workforce management interface designed to support location-aware attendance, administrative workflows, and operational monitoring across multi-branch organizations.

The application provides administrators with a clear, responsive, and role-aware dashboard to manage workforce activities efficiently.

**Built to simulate real-world enterprise UI requirements with a focus on usability, performance, and scalable component structure.**

## Key Features

- Role-based interface rendering  
- Secure authentication flows  
- Real-time attendance visibility  
- Administrative dashboards  
- Leave request management  
- Policy-aware UI behavior  
- Responsive layout for multiple screen sizes  
- Map-based branch visualization using React Leaflet  
- Biometric-supported attendance workflows  

---

## Tech Stack

- React (Vite)  
- JavaScript (ES6+)  
- Context API  
- React Router  
- RESTful APIs  
- React Leaflet  
- Bootstrap  

---

## Architecture Approach

The frontend follows a modular, component-driven architecture to promote maintainability and scalability.

- Reusable UI components  
- Separation between business logic and presentation  
- Centralized API handling  
- Structured routing  

---

## Performance Considerations

- Optimized component rendering  
- Efficient API consumption  
- Lazy loading where applicable  
- Clean state management

## Getting Started

### Prerequisites
- Node.js  
- npm  

### Installation

```bash
git clone https://github.com/Noura-Elbrashy/WorkGuard-frontend.git
cd WorkGuard-frontend
npm install
npm run dev
```
---
## Environment Variables

Create a `.env` file in the root directory and configure the following variables:

VITE_API_URL=

NODE_ENV=
